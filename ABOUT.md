# About Minty Host Guest Manager

## Overview
Minty Host Guest Manager is a modern web application designed for managing bookings and guest data in touristic apartments. It provides an intuitive interface for property managers to handle reservations, guest information, and operational workflows efficiently.

## Key Features
- **Booking Management**: Comprehensive booking tracking with status updates
- **Guest Database**: Complete CRUD operations for guest records
- **Search & Filter**: Real-time search functionality across bookings and guests
- **Responsive Design**: Modern UI built with Vue.js and Tailwind CSS
- **API-Driven**: RESTful API architecture with Laravel backend

## Technology Stack
- **Backend**: Laravel 10 (PHP)
- **Frontend**: Vue.js 3 with Inertia.js
- **Database**: SQLite (development), configurable for production
- **Styling**: Tailwind CSS
- **Build Tool**: Vite

## Purpose
This application streamlines the management of short-term rental properties by providing property owners and managers with the tools needed to:
- Track booking lifecycles
- Maintain accurate guest records
- Improve operational efficiency
- Enhance guest experience through better data management

## Development Status
Currently in active development with ongoing feature enhancements and testing coverage.
